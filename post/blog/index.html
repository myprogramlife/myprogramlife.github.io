Apr 22, 2020
1 min read

我的第一篇博客

四句搭建自己的博客：

一、创建站点

hugo new site myblog

二、下载博客主题

git clone git@github.com:shankar/hugo-grapes.git

三、新建博客文章

hugo new first.md

四、启动hugo

hugo server -t 主题文件夹名 --buildDrafts
